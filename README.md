# SmartHR

SmartHR is a Human Resource Management System built with React and Firebase to simplify day-to-day HR operations. The application provides a centralized platform for managing employees, attendance, leave requests, authentication, and role-based access control through an intuitive and responsive interface.

## Features

- Authentication & Role-Based Access Control
- Employee Management
- Attendance Tracking
- Leave Management
- Dashboard & Analytics
- Company Settings

## Tech Stack

- React
- Firebase
- CSS
- React Router
